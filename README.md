## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript repository for Identity Reconciliation.

Node Version 18.15.0

## Tech Stack

NestJs
MySQL

## Installation

```bash
$ yarn install
```

## Running the app

Create a .env file in the root folder (Take reference from .env.example file)

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
